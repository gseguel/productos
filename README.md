Se debe ejecutar proyecto con Spring boot
una vez inicializado se pueden realizar pruebas del back con la siguiente URL de swagger o con postman
http://localhost:8080/swagger-ui.html


Se realizo proyecto con:
Maven
MVC/springboot
Java 11

para persistencia se utilizo MyIbatis ya que ahora mucho codigo JDBC
Se añadio Swagger
La arquitectura se encuentra definida en:
config(configuracion de swagger)
Controller
Service
Repository
model
exceptions (Se crearon excepciones para errores mas descriptivos)
